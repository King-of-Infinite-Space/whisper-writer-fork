## Whisper Writer <img width="40" height="40" src="./assets/ww-logo-new.png" align="right" >


- This is a fork of [dariox1337/whisper-writer](https://github.com/dariox1337/whisper-writer)
  - which is a fork of [savbell/whisper-writer](https://github.com/savbell/whisper-writer)

I'm not entirely satisfied with the accuracy of Windows built-in voice typing, so I'm using this as an alternative. [LazyTyper](https://github.com/oldcai/LazyTyper-releases) is another good alternative but closed source. I also had trouble making it work on Linux.

### Note for Linux
If you are using the `evdev` input backend or `uinput` output method, you may need to set up `uaccess` rules, otherwise they require `sudo`.
